## Commands

### 1. Daily Report
```bash
cost-audit report
```
Shows:
- Total cost for today
- Breakdown by operation (code_generation, debugging, etc.)
- Breakdown by model (Claude Opus, Sonnet, Haiku)
- Average cost per operation
- Weekly forecast

**Output Example:**
```

======================================================================

  • code_generation         $  0.0675 ( 17.6%)

🤖 Cost by Model:
  • claude-opus-4-8         $  0.3825 (100.0%)

  Daily Average:     $0.3825
```

### 2. Track Manual Operation
```bash
cost-audit track "code_generation" 2000 500
```

Tracks:
- **Operation type**: code_generation, debugging, testing, etc.
- **Input tokens**: Tokens sent to Claude
- **Output tokens**: Tokens generated by Claude

**Output Example:**
```
✅ Tracked operation: code_generation
   Cost: $0.0675
```

### 3. Weekly Forecast
```bash
cost-audit forecast
```

Shows:
- Daily average spend
- Weekly estimate
- Monthly estimate
- Based on last 7 days of activity

**Output Example:**
```
📅 Weekly Forecast:
  Daily Average:    $0.3825
  Weekly Forecast:  $2.68
  Monthly Forecast: $11.48
```

### 4. Budget Alerts
```bash
cost-audit alerts
```

Checks:
- Daily budget threshold (75% = $37.50 if monthly budget is $50)
- Monthly forecast vs budget
- Cost anomalies

**Output Example:**
```
⚠️  Active Alerts:
  [MEDIUM] Daily spend ($0.38) is 0% of daily budget ($50.00)
```

## Automatic Tracking

The skill automatically tracks operations via Claude Code hooks in `.claude/settings.json`:

- ✅ Automatically logs `Edit`, `Write`, `Bash`, `Read` operations
- ✅ Estimates tokens from operation size
- ✅ Calculates cost in real-time
- ✅ Stores in `~/.pycostaudit/skill_data.json`

No manual tracking needed!

---

## Data Storage

All cost data is stored locally in:
```
~/.pycostaudit/skill_data.json
```

This JSON file contains:
- Session ID
- List of all tracked operations
- Timestamps for each operation
- Token counts and costs

---

## Pricing (Claude Opus 4.8)

Default pricing used:
- **Input**: $15.00 per 1M tokens
- **Output**: $75.00 per 1M tokens

To use different pricing, edit the skill:
```python
# In pycostaudit_skill.py, line 64-65
input_cost = (tokens_input / 1_000_000) * 15.00  # Change here
output_cost = (tokens_output / 1_000_000) * 75.00  # And here
```

## Troubleshooting

### No operations showing?
1. Check that operations are being performed (Edit, Write, Bash, Read)
2. Verify `.claude/settings.json` exists and hook is configured
3. Manually track with: `cost-audit track "operation" 2000 500`

### Wrong cost calculation?
1. Verify Claude model pricing in skill
2. Check token counts are reasonable
3. Compare to Anthropic pricing: https://www.anthropic.com/pricing

### Data not persisting?
1. Check `~/.pycostaudit/` directory exists
2. Verify write permissions: `ls -la ~/.pycostaudit/`
3. Try creating manually: `mkdir -p ~/.pycostaudit`
